# BigInt Library (Учебный проект)

Учебная библиотека `bigint.h` и `bigint_extra.h` для работы с большими целыми числами в C++.  
Проект создан для изучения алгоритмов работы с большими числами и их свойств.

---

## Основной тип: `bigint`

`bigint` — тип для хранения больших целых чисел. Основные характеристики:

- **BASE**: 10^9 (каждый элемент массива `digits` хранит 9 цифр).  
- **Знак числа**: `isNegative()` — проверка, является ли число отрицательным.  
- **Количество блоков цифр**: `size()` — число блоков по BASE.  
- **Количество десятичных цифр**: `num_digits()` — точное количество цифр числа.  
- **Модуль числа**: `abs()` — абсолютное значение числа.  
- **Строковое представление**: `to_string()`.  
- **Целочисленный квадратный корень**: `sqrt()` — наибольшее целое `r`, такое что `r*r <= n`.

---

## Арифметика

- **Сложение и вычитание** — стандартные алгоритмы по массивам цифр.  
- **Умножение**:
  - Небольшие числа — обычное умножение по разрядам.  
  - Большие числа — FFT (быстрое преобразование Фурье) для ускорения.  
  - Очень большие числа — включается алгоритм Карацубы (Karatsuba). Для экстремально больших чисел лучше использовать GMP.  
- **Деление и остаток**:
  - `divmod(a, b)` возвращает пару `(a / b, a % b)`.  
  - Оптимизация остатков по 2, 5 и другим базам для ускорения.

---

## GCD и EGCD

- `gcd(a, b)` — наибольший общий делитель. Используется **бинарный алгоритм Евклида (Binary GCD)**.  
- `egcd(x, y)` — расширенный бинарный алгоритм Евклида.  
  Возвращает `{C, D, g*v}`: коэффициенты Безу и GCD.  
  Не работает с нулём.  
- `modular_inverse(x, y)` — обратный элемент по модулю `y`. Возвращает `-1`, если обратного не существует.

---

## Возведение в степень по модулю

- `pow_mod(n, power, m)` — быстрое возведение в степень по модулю (modular exponentiation).

---

## Генерация случайных чисел (пробные версии)

- `random_bigint()` — случайное большое число.  
- `random_bigint(length)` — случайное число заданной длины (в блоках BASE).  
- `random_prime_bigint()` — случайное большое простое число.  
- `random_prime_bigint(n)` — случайное простое число с n блоками.  

Эти функции пока медленные и предназначены для учебных целей.

---

## Проверка простоты

- `MillerRabbin(n)` — проверка числа на простоту детерминированным тестом Миллера-Раббина.  
- `test_if_prime(a)` — проверка с использованием малых простых чисел и Миллера-Раббина.  
- `Fermat(n, tries)` — тест простоты Ферма.

---

## `bigint_extra.h` — расширенные функции

### Алгоритмы работы с простыми числами

- **sieve_mil()** — решето Эратосфена для всех простых до 1 000 000.  
- **sieve_int()** — расширенное решето для int32_t диапазона.  
- **prime_count(n)** — количество простых чисел меньше `n`.  
- **MillerRabbin** — проверка на простоту (для int32_t и bigint).  

### Факторизация и генерация простых чисел

- `random_prime_bigint()` — генерация случайного простого числа (пробная версия).  
- `random_prime_bigint(n)` — генерация случайного простого числа заданной длины.  

### Комбинаторика

- **factorial(n)** — факториал для int и bigint.  
- **binomial(n, k)** — биномиальные коэффициенты.  
- **partperm(n, k)** — частичные перестановки.  
- **fibonacci(n)** — вычисление n-го числа Фибоначчи.

---

## Примечания

- Библиотека предназначена для **учебных и экспериментальных целей**.  
- Для очень больших чисел и производственных задач рекомендуется использовать **GMP** или другие проверенные библиотеки.  
- Алгоритмы для работы с простыми числами и генерации случайных чисел подходят для средних по размеру чисел.

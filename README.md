# BigInt Library (Учебный проект)

Учебная библиотека `bigint.h` и `bigint_extra.h` для работы с большими целыми числами в C++.  
Используется собственная реализация типа `bigint` с базой `BASE = 10^9`.  
Проект носит учебный характер, все функции созданы для практики и экспериментов.

---

## Основной тип: `bigint`

`bigint` — тип для хранения больших целых чисел. Основные характеристики:

- **BASE**: 10^9 (каждый элемент массива `digits` хранит 9 цифр).
- **Знак числа**: хранится отдельно (`isNegative()`).
- **Количество цифр**: `size()` возвращает число "блоков" по BASE, а функция `num_digits()` — точное количество десятичных цифр.
- **Модуль**: `abs()` возвращает абсолютное значение числа.
- **Строковое представление**: `to_string()` — конвертация числа в строку.
- **Целочисленный квадратный корень**: `sqrt()` возвращает наибольшее целое `r`, такое что `r*r <= n`.

---

## Арифметика

- **Сложение и вычитание** — выполняется по обычным алгоритмам работы с массивами цифр.
- **Умножение**:
  - Небольшие числа — стандартное умножение по разрядам.
  - Большие числа — FFT (быстрое преобразование Фурье) для ускорения.
  - Очень большие числа — включается алгоритм Карацубы (Karatsuba). ⚠️ Для экстремально больших чисел лучше использовать библиотеки вроде GMP.
- **Деление и остаток**:
  - `divmod(a, b)` возвращает пару `(a / b, a % b)`.
  - Оптимизация остатков по 2, 5 и другим базам для ускорения.

---

## GCD и EGCD

- `bigint gcd(const bigint& a, const bigint& b)` — наибольший общий делитель.  
  ⚡ Используется **бинарный алгоритм Евклида** (Binary GCD), который быстрее стандартного для больших чисел.

- `std::vector<bigint> egcd(bigint x, bigint y)` — расширенный бинарный алгоритм Евклида.  
  Возвращает `{C, D, g*v}`: коэффициенты Безу и gcd.  
  ⚠️ Не работает с нулём.

- `bigint modular_inverse(bigint x, bigint y)` — обратный элемент по модулю `y`.  
  Возвращает `-1`, если обратного не существует.

---

## Генерация случайных чисел (пробные версии)

- `random_bigint()` — случайное большое число.  
- `random_bigint(int length)` — случайное число заданной длины (в блоках BASE).  
- `random_prime_bigint()` — случайное большое простое число.  
- `random_prime_bigint(int n)` — случайное простое число с n блоками.  

⚠️ Все эти функции пока **медленные и экспериментальные**, предназначены для учебных целей.

---

## Простейшие функции из `bigint_extra.h`

- **Факториалы**: `factorial(int n)`, `factorial(bigint n)`  
- **Биномиальные коэффициенты**: `binomial(int n, int k)`  
- **Частичные перестановки**: `partperm(int n, int k)`  
- **Фибоначчи**: `fibonacci(int n)`

---

## Примечания

- Библиотека предназначена для учебных и экспериментальных целей.  
- Для очень больших чисел и производственных задач рекомендуется использовать GMP или другие проверенные библиотеки.  
- Алгоритмы для работы с простыми числами, генерации случайных чисел и проверки простоты (`MillerRabbin`, `Fermat`) — также учебные и медленные.
